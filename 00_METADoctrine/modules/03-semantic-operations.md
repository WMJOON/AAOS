# Semantic Operations

## 정의

Semantic operation은 실행 성공률이 아니라, 판정 가능성과 책임 추적 가능성을 보장하는 연산이다.

## 운영 원칙

1. Judgability-first: 판단 근거를 먼저 남긴다.
2. Local chart ownership: 각 레이어는 자기 책임 경계를 문서로 명시한다.
3. Evidence before action: 실행 전 증빙 경로를 확보한다.
4. Entropy budget: 설명 비용이 통제 비용을 넘으면 설계를 축소한다.

## 엔트로피 가드레일

- 프론트매터 키 수 최소화
- 규칙 문서 분할(코어/세부)
- 레거시 제도 재도입 금지
