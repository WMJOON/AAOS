# 00_METADoctrine (Draft/Planning Workspace)

이 폴더는 **Draft/Planning 레이어(비정통/비집행)** 를 위한 작업대다.
운영상 이 폴더는 사람 직접 관리 구획이 아니라, 인간이 정의한 METADoctrine 규범 하에서
Agent가 구성하는 사전 작업 구역이다.

- Canonical source of truth: `../README.md`, `./DNA.md`
- Draft는 **집행 권한이 없으며**, Canonical 문서의 **normative reference(규범 참조)** 가 될 수 없다.
- Draft는 만료(`time_bound.expires`)를 가져야 하며, 기본 TTL(권장)은 30일이다.
