# MANIFEST.sha256 (template)
#
# 목적: 패키지 내부 파일의 sha256 목록을 고정해 무결성/재현성을 강화한다.
# 규칙: append-only 패키지에서는 파일을 수정하지 않는다. (수정이 필요하면 “새 패키지”를 만든다.)
#
# 생성 예시(권장):
# 1) 패키지 루트에서:
#    find payload -type f -print0 | sort -z | xargs -0 shasum -a 256 > MANIFEST.sha256
#
# 2) 매니페스트 자체의 해시(인덱스/원장에 기록):
#    shasum -a 256 MANIFEST.sha256
#
# 아래는 “sha256  filepath” 형식 예시:
#
# <sha256>  payload/<file1>
# <sha256>  payload/<file2>
