# AAOS Nucleus

Nucleus는 AAOS의 **기관 레이어(Validation Engine)** 이다.

- 직접 실행하지 않는다(Non-Execution).
- 기록(Archive), 판정/집행(Immune), 합의/근거 구조화(Deliberation)를 통해 규범을 진화시킨다.

## 구성

- `Record_Archive/`
- `Immune_system/`
- `Deliberation_Chamber/`
