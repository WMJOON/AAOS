# Rule 03: Permission Principal

- 다음 행위는 심판 전제:
  - Tool/API 접근
  - 구조 생성/확장
  - 영구 보관 요청
  - 규칙/skill 변경으로 계보 영향 발생
- 판정 결과:
  - Canonical / Canonical-Conditional / Non-Canonical
- 요청 스키마 기준:
  - requester, action, target, risk_level, justification
  - time_bound.expires
  - natural_dissolution.termination_conditions / dissolution_steps
