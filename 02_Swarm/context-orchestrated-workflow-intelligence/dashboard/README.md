# Production Dashboard Outputs

This directory stores portable dashboard outputs generated by:

- `02_Swarm/context-orchestrated-workflow-intelligence/scripts/build_production_dashboards.py`

Output files:

- `production-proposals.json`
- `production-proposals.csv`
- `user-inbox.json`
- `user-inbox.csv`
- `archived-proposals.json`
- `archived-proposals.csv`

Filtering policy:

- Production view: `proposal_status != closed` and `visibility_tier == must_show`
- Approval queue: `hitl_stage == approval_required` and `checked == true`
- User inbox: `must_show` and `user_action_required == true` and not closed
